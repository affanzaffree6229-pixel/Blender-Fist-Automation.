# Blender-Fist-Automation.
it automates fist movements
Blender Fist Automation Script
This script adds a "Fist" slider to any character's hand bone in Blender.
How to use:
Open the Scripting tab in Blender.
Paste the code and hit Run Script.
In Pose Mode, select the Hand Bone.
Press N to open the sidebar and look for the Fist tab.
Move the slider to clench the hand!
